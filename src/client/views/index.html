Sentiment Evaluation App

This web application allows you to run Natural Language Processing (NLP) on articles or blogs found on other websites. Using Sentiment Analysis, you will receive pertinent information about the article, like whether the content is subjective (opinion) or objective (fact-based) and whether it is positive, neutral, or negative in tone.
